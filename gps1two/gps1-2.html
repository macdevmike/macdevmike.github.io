GPS Topics

Pair programming is concept of having two individuals working on the same code or coding project with one as the driver, and one as the navigator. Pair programming emphasises the concept of pair programming and collaboration.

Pair programming is important for several reasons when it comes to DBC. First and foremost getting a second set of eyes and a new brain to look over what you are working on. It also gets you into the practice of working with others when collaborting on programs or projects.
